# Paycheck Budget

Next.js budget planner for paycheck-by-paycheck control:
- Paycheck A/B breakdown
- Safe-to-spend guardrails
- Debt payoff mode and debt-free mode
- Debt progress tracking and payoff estimate

## Local run

```bash
npm install
npm run dev
```

## Deploy on Vercel

1. Push this repo to GitHub.
2. Import the repo in Vercel.
3. Keep defaults (Framework: Next.js).
4. Deploy.

No environment variables are required.
